# Ecommerce-proyecto-final-23040466---23040429
Proyecto final Ecommerce 23040466 - 23040429

El frontend solo necesita descargar Angular debido a que estando en el archivo lo hacía muy pesado, para hacerlo ejecutar solo es necesario descargarle Angular.

El proyecto consiste en un E-commerce de una plataforma de venta de videojuegos para PC, los usuarios pueden buscar que tipo de videojuego quieren en base a un filtro de busqueda de nombre o de busqueda de categoría, además de que para comprar un juego es necesario tener una cuenta creada, una vez se compra un juego se envía automaticamente al carrito donde se estará esperando la confirmación del usuario o la cancelación antes de la compra.
Los videojuegos tienen sus nombres, descripciones, requisitos de pc minimos, logotipo y precio.
El usuario tiene la capacidad de crearse una cuenta, iniciar sesión en esa cuenta, interactuar con el menú, ver opciones de juego y comprarlos (confirmando su compra o si fue un error o hubo cambio de opinion la pueden borrar).
